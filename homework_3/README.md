#运行方法               
将两文件放在同一文件夹中，命令行进入该文件夹，最大化命令行窗口后`python homework_3.py`运行            
Windows用户请修改`MiniGUI.py`下的`os.sys('clear')`为`os.sys('cls')`后运行
